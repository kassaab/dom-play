About

This interactive web page demonstrates DOM manipulation using plain JavaScript. It allows actors in a play to tap their character’s name to highlight all their lines — making it easier to rehearse.

The page works dynamically (without reloading) and handles any number of actors and script lines.

The page uses only built-in JavaScript and custom HTML attributes for flexibility and interactivity.
